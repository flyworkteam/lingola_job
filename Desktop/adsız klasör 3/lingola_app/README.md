# Lingola

Lingola - Dil öğrenme uygulaması.

## Gereksinimler

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (^3.10.7)
- Dart SDK ^3.10.7

## Kurulum

```bash
flutter pub get
```

## Çalıştırma

```bash
# Cihaz listesi
flutter devices

# Uygulamayı çalıştır (açık emülatör veya cihaz)
flutter run
```

## Proje yapısı

- `lib/main.dart` — Uygulama giriş noktası
- `pubspec.yaml` — Bağımlılıklar ve proje ayarları
- `analysis_options.yaml` — Dart/Flutter lint kuralları

## Build

```bash
# Android APK
flutter build apk

# iOS (macOS gerekli)
flutter build ios

# Web
flutter build web
```
